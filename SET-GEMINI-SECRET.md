# Set GEMINI_API_KEY on Supabase

Run this ONE command in your terminal (replace the key with your real one):

```bash
npx supabase secrets set GEMINI_API_KEY=your_actual_key_here --project-ref syoungrummuurrptckbt
```

Get your free Gemini API key at: https://aistudio.google.com/app/apikey

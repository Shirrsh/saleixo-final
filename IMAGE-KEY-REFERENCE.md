# Saleixo — Image & Video Key Reference

Use this as your upload checklist. For each asset:
1. Upload the file to Supabase Storage (`site-assets` bucket) in the matching folder
2. Go to `/admin/images` (or `/admin/videos`) and add an entry with the exact `image_key` below
3. The site will automatically use your new asset instead of the placeholder

## HOW TO USE A VIDEO IN A COMPONENT

```tsx
import { useSiteVideos } from '@/hooks/useSiteVideos';

const MyComponent = () => {
  const { getVideoUrl, getPosterUrl, getAltText } = useSiteVideos('testimonials');

  return (
    <video
      src={getVideoUrl('testimonial_james_whitfield')}
      poster={getPosterUrl('testimonial_james_whitfield')}
      aria-label={getAltText('testimonial_james_whitfield', 'James Whitfield testimonial')}
      autoPlay
      muted
      loop
      playsInline
      className="w-full h-full object-cover"
    />
  );
};
```
